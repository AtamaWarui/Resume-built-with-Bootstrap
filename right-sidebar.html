Project 1

Academic project : Cooperating and self-organized robotic systems

Cooperating and self-organized robotic systems : Solving an application problem using the contraint programming platform for the mobile multi-robots application problem.

We provide an example application that illustrates how our platform can be used to solve multi-robot problems using constraint programming techniques with the ROS system and MORSE simulator tool.

3D problems n² queens drone robots application

The problem focuses on cooperating robotic systems based on the study of collective and collaborator decision problems. To make this concrete, we thought of an autopilot application of individual vehicles based on simplifying assumptions such as modeling and solving 3D problems n² queens drone robots (quadrotor dynamic) programmed in the operating system ROS that control in an uncertain environment at the MORSE simulator.
